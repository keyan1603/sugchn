How to update the live site:
1. Click Generate data.js below to see the updated data file content.
2. Copy the entire content.
3. Go to your GitHub repository, open data.js, and paste to replace it.
4. Commit the change — GitHub Pages will automatically update the live site within 1-2 minutes.

To change your admin passphrase:
1. Go to SHA-256 generator
2. Type your new passphrase → copy the hash
3. Open admin.html in your repo and replace the value of PASSPHRASE_HASH
4. Commit — your actual passphrase is never stored in the file.
